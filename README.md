# Dokku timezone

Set timezone on buildstep image.

## Requirements

* Dokku 0.4.0+

## Installation

```
dokku plugin:install https://github.com/leominov/dokku-timezone.git dokku-timezone
```

## Hooks

This plugin provides hooks:

* `pre-build-buildpack`
